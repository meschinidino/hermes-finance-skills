# C-5 Edge & Cruxes Prompt Contract

Draft an evidence-backed edge statement for Senior review. The output is not final and must remain ratifiable.

Required areas:
- a no-trade steelman;
- a non-trivial counterparty;
- structural mispricing, or explicit no-structural-edge/pass framing;
- variant view, or explicit fairly-priced/pass framing;
- concrete catalysts;
- exactly three `edge_crux` records when an edge is asserted;
- optional `pass_falsifier` records when no structural edge exists; do not manufacture them, but any filed pass-falsifier must include metric, threshold direction, threshold value, and check-by date.

This prompt carries no enforcement weight. The Python audit rejects unsupported or malformed drafts.
